# Project 4 - OCR Text Recognition

## Objective
Develop an Optical Character Recognition (OCR) system that extracts text from an image using Python.

## Technologies Used
- Python
- OpenCV
- Pytesseract

## Features
- Loads an image from the project folder
- Converts the image to grayscale
- Applies Gaussian Blur for noise reduction
- Uses thresholding to improve text clarity
- Extracts text using Tesseract OCR
- Displays the recognized text in the terminal
