# DecodeLabs AI Internship - Project 1

## Project Title

Rule-Based AI Chatbot.

## Objective

Develop a simple rule-based chatbot that responds to user queries using predefined rules and pattern matching.

## Technologies Used

* Python
* Regular Expressions (re)
* Random Module
* Datetime Module

## Features

* Handles greetings
* Answers common questions
* Displays current date and time
* Tells jokes
* Motivational responses
* Coin flip and Rock-Paper-Scissors
* Cleans user input using Regular Expressions
* Exit functionality

## How to Run

1. Open the project in Python or VS Code.
2. Run `main.py`.
3. Type your messages in the terminal.
4. Type `bye` to exit.
