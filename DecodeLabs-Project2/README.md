# DecodeLabs AI Internship - Project 2

## Project Title

Data Classification Using AI (Iris Dataset)

## Objective

Develop a machine learning model to classify Iris flowers into three species using the K-Nearest Neighbors (KNN) algorithm.

## Technologies Used

* Python
* Scikit-learn
* NumPy

## Algorithm

* K-Nearest Neighbors (KNN)

## Features

* Load the Iris dataset
* Split data into training and testing sets
* Scale features using StandardScaler
* Train the KNN model
* Predict flower species
* Display Accuracy
* Display Confusion Matrix
* Display Classification Report
* Display F1 Score

## Output

* Accuracy: 100%
* F1 Score: 1.0
