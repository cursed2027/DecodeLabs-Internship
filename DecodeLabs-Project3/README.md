# DecodeLabs AI Internship - Project 3

## Project Title

Tech Stack Recommendation System

## Objective

Develop an AI-based recommendation system that suggests suitable technology career paths based on a user's skills and interests.

## Technologies Used

* Python
* Scikit-learn

## Algorithms Used

* TF-IDF Vectorizer
* Cosine Similarity

## Features

* Accepts three user skills/interests
* Converts text into TF-IDF vectors
* Calculates similarity between user skills and predefined job roles
* Sorts career matches by similarity score
* Displays the Top 3 recommended career paths

## Example Output

Input Skills:

* Python
* Machine Learning
* Pandas

Recommended Careers:

1. Data Scientist
2. AI Engineer
3. Backend Developer
